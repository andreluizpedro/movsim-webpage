Download the the movsim viewer. Start the application with "java -jar viewer-1.2-jar-with-dependencies". You need to have Java installed to run movsim.

A traffic scenario is defined with a xml configuration file. More examples can be found on the github page.. Download the xml files and the corresponding road network files (xodr) and open the xml file within the application. For advanced usage we recommand to check out the wiki more closely.
