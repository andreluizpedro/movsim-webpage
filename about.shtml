About MovSim (Multi-model open-source vehicular-traffic Simulator)

MovSim is a microscopic road traffic simulator. The simulator implements different car-following and model categories like time-continuous models, iterated maps and cellular automata. Recently, we added a microscopic model for cross-country skiing. Lane changes are modeled with the general MOBIL strategy based on accelerations.

MovSim is intended as a tool for model investigation and comparison. Our goal is to model and simulate all basic traffic situations and discrete decision makings like lane changes, reacting to a traffic light, yielding, overtaking on rural roads etc. in a generic way for the implemented models.

MovSim can be run from commandline or with a graphical user interface with visualization. Several output quantities can be written to file for further analysis. MovSim also provides a physics-based fuel-consumption model to calculate consumption on an individual or collective level.

MovSim has been initiated by Arne Kesting in June 2011. Further contributors are Martin Budden, Ralph Germ and Martin Treiber.

MovSim is an open-source project licensed under the GPLv3. The software is written in Java and hosted on Github.
